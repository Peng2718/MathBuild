## 一、灰色关联分析完整代码
```python
import numpy as np
import pandas as pd

def grey_relation_analysis(data, ref_col=0, rho=0.5):
    """
    灰色关联分析（GRA）实现
    :param data: 输入数据矩阵 (样本数, 指标数)，二维列表/数组
    :param ref_col: 参考序列所在列（默认第0列）
    :param rho: 分辨系数（通常取0.5）
    :return: 各指标与参考序列的关联系数矩阵、各指标灰色关联度
    """
    # 1. 数据标准化（消除量纲，采用均值化）
    data = np.array(data, dtype=float)
    n, m = data.shape  # n=样本数，m=指标数
    
    # 均值化标准化（避免极值影响，灰色关联常用）
    mean_vals = np.mean(data, axis=0)
    std_data = data / mean_vals  # 标准化后矩阵

    # 2. 确定参考序列（默认选第0列，也可自定义）
    ref_series = std_data[:, ref_col]
    ref_series = ref_series.reshape(-1, 1)  # 转为列向量，方便广播计算

    # 3. 计算差序列（参考序列与各指标序列的绝对差）
    diff_series = np.abs(std_data - ref_series)

    # 4. 找两级最大差、最小差
    min_diff = np.min(diff_series)
    max_diff = np.max(diff_series)

    # 5. 计算关联系数
    correlation_coeff = (min_diff + rho * max_diff) / (diff_series + rho * max_diff)

    # 6. 计算灰色关联度（关联系数的均值）
    correlation_degree = np.mean(correlation_coeff, axis=0)

    return correlation_coeff, correlation_degree

# ------------------- 实战示例 -------------------
if __name__ == "__main__":
    # 沿用之前的学生成绩数据：3学生，3指标（数学、英语、体育）
    # 假设以"数学"为参考序列，分析英语、体育与数学的关联程度
    score_data = [
        [80, 90, 70],
        [90, 85, 80],
        [70, 80, 90]
    ]
    cols = ["数学（参考）", "英语", "体育"]

    # 执行灰色关联分析
    coeff, degree = grey_relation_analysis(score_data, ref_col=0)

    # 格式化输出结果
    print("===== 原始数据 =====")
    print(pd.DataFrame(score_data, columns=cols))

    print("\n===== 关联系数矩阵 =====")
    coeff_df = pd.DataFrame(coeff, columns=cols)
    print(coeff_df.round(4))

    print("\n===== 灰色关联度 =====")
    degree_df = pd.DataFrame({
        "指标": cols,
        "关联度": degree.round(4)
    })
    print(degree_df)

    # 关联度排序（越大越相关）
    sorted_idx = np.argsort(degree)[::-1]
    print("\n===== 关联度排序 =====")
    for i, idx in enumerate(sorted_idx):
        print(f"第{i+1}位：{cols[idx]} (关联度：{degree[idx]:.4f})")
```

## 二、代码关键部分解释
1. **数据标准化**：  
   用**均值化标准化**（`数据/列均值`），而非min-max，这是灰色关联分析的常用做法，能避免单一极值对结果的过度影响。
   
2. **参考序列选择**：  
   `ref_col=0` 表示选第0列（数学）作为参考，你可以根据需求修改（比如想分析所有指标与“综合成绩”的关联，可先计算综合成绩作为新列）。

3. **分辨系数rho**：  
   通常取0.5，rho越小，区分能力越强；rho越大，区分能力越弱，0.5是行业通用值。

4. **关联系数与关联度**：  
   关联系数是“每个样本”的关联程度，关联度是关联系数的均值，代表“整体”关联程度，**关联度越大，说明该指标与参考序列的相关性越强**。

# 假设已用熵权法算出权重w = [0.31, 0.23, 0.46]
